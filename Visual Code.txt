Visual Code-da IDE  yoxdur, visual studio code text-editordur. Atom Text editorunun
təməlində duran Electron node,js,HTML,CSS,javascript kimi standartlar cross-platform tətbiqləri təkmilləşdirməyə imkan verir. Burada yalnız 
.NET tətbiqlər deyil, hər cür text yönümlü kodu yazmaq mümkündür.
Visual Studio Code ilə Debug etmək mümkündür. Amma OS X və Unixdə hələki CoreCLR və ASP.NET 5 tətbiqlərini debug etmək mümkün deyil.
Tətbiqləri çalışdırmaq üçün dnx əmrlərindən istifadə olunur. Command Pallette(F1) köməyilə istədiyimiz əmrləri çalışdıra bilərik. Visual Code da
GitHub dəstəyi də var.
